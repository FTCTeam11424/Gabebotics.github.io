History of logic

Logic Developed Over Time
Aristotle wrote The Organon, the first ever logic textbook (400 BC)
Sir Francis Bacon published Novum Organon Scientarium (1620 AD)
  • Bacon develops inductive reasoning, experiment before principle
Bertrand Russel and Alfred North Whitehead published Principia Mathematica which established symbolic logic (1913)

The Philosophical Importance of the Two Logics
Aristotelian logic asserts several presuppositions - the world is real, we can know it thrugh the senses, and we can discover true about the real world.
  • Logic tells us true things about the real world

Symnbolic logic makes no such claim - focused on the manipulation of symbols in correct ways (P -> Q)
  • Symbolic logic takes the most important arguments and makes them into symbols in order to distance us from the actual argument

Aristotelian logic uses words(Beowulf), Symbolic logic is a mathematical logic(Instruction manual).
